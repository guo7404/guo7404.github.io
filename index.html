안녕하세요! 저는 사용자 중심의 웹 서비스를 만드는 것을 좋아하는 백엔드 개발자입니다. 효율적인 시스템 설계와 안정적인 서비스 운영에 관심이 많으며, 새로운 기술을 배우고 적용하는 것을 즐깁니다.

주요 관심 분야:

  • 효율적인 백엔드 시스템 설계
  • 데이터베이스 최적화
  • RESTful API 설계
  • 보안 및 성능 개선
